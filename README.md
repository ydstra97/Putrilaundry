- Tombol cetak struk

## Cara online lewat GitHub Pages

1. Buat repository baru di GitHub dengan nama `putri-laundry`.
2. Upload semua file di folder ini ke repository.
3. Buka Settings > Pages.
4. Pada Build and deployment, pilih Source: Deploy from a branch.
5. Pilih Branch: main dan Folder: /root.
6. Klik Save.
7. Tunggu beberapa menit, lalu buka link GitHub Pages yang muncul.

Catatan: fitur upload bukti pembayaran di website statis ini hanya menampilkan preview di halaman. Agar bukti benar-benar tersimpan ke server/database, perlu backend/Firebase/Google Form.
